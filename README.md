### Project Overview
This project analyzes customer service interactions from a call center dataset to understand customer sentiment, service performance, and operational efficiency.
The dashboard provides insights into how effectively customer issues are handled, helping identify patterns that influence customer satisfaction (CSAT) and response performance.
The analysis was built in Microsoft Excel using pivot tables and interactive dashboard elements.









### Project Objectives
The main goals of this project are to:
- Monitor customer satisfaction (CSAT)
- Analyze customer sentiment trends
- Evaluate call center performance against SLA targets
- Identify the most common customer issues
- Understand how channels and locations influence service performance


### Dataset
The dataset contains customer service interaction records including but not limited to:
- Customer ID
- Customer Name
- Sentiment (Very Positive, Positive, Neutral, Negative, Very Negative)
- CSAT Score
- Call Timestamp


### Dashboard Feautures
The dashboard provides a visual summary of key call center metrics including:

1. Customer Sentiment Analysis
   - Tracks the distribution of customer sentiment to understand overall customer experience.


2. Customer Satisfaction (CSAT)
   - Measures satisfaction scores given by customers after service interactions.

3. Response Time Performance
   - Evaluates whether calls are handled within or above SLA, indicating service efficiency.


4. Call Reasons
   - Highlights the most common reasons customers contact support, such as:
       - Billing questions
       -  Service outages
       -  Payments
       -  Other support requests
  
5. Communication Channels
   - Compares interactions across service channels such as:
      -  Call Center
      -  Chatbot
      -  Web
  
6. Geographic Insights
   - Displays call data based on cities, states, and call center locations.


7. Call Duration Analysis
   - Provides insights into how long calls typically last and how that relates to service outcomes.



### Tools Used
